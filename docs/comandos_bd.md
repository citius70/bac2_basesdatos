# Comandos básicos SQL

Una vez dominado la identificación y uso de** claves primarias** y **foráneas** en el diseño de tablas y relaciones, el siguiente paso lógico es introducir las **operaciones fundamentales con SQL** para manipulación y consulta de datos. Esto permite poner en práctica el diseño lógico realizando acciones sobre los datos reales.

 Estas operaciones son realizadas una vez ejecutados los **comandos** SQL. Operaciones como *crear/eliminar una base de datos, crear/editar/eliminar una tabla, crear/editar/eliminar un registro de la tabla, filtrar ciertos registros que cumplan una o varia condiciones*..., llevan asociadas sus corrrespondientes comandos.

Los comandos de SQL se agrupan en distintos tipos, según la función que cumplen dentro de la gestión y manipulación de bases de datos.

## Tipos principales de comandos SQL

Los comandos más importantes de SQL se agrupan en tres tipos, según la función que cumplen dentro de la gestión y manipulación de bases de datos.

| Tipo (sigla) | Nombre | Función principal | Comandos clásicos |
|--------------|--------|------------------|------------------|
| **DDL**          | Data Definition Language (Definición de datos) | Crear, modificar y borrar estructuras de la base (tablas, campos, etc. | CREATE, ALTER, DROP |
| **DML**          | Data Manipulation Language (Manipulación de datos) | Insertar, actualizar, borrar datos almacenados | INSERT, UPDATE, DELETE |
| **DQL**          | Data Query Language (Consulta de datos) | Consultar y extraer datos | SELECT |
