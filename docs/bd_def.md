# Definición de bases de datos

Las bases de dato  son esenciales porque nos ayudan a resolver problemas reales y a tomar decisiones informadas en ámbitos como la tecnología, la ciencia, el comercio o la educación.

![bd](img/biblioteca.png){ width="40%" style="float: right; margin-left: 1em;" }

!!! note "Bases de datos"

    Una base de datos es un sistema que permite almacenar, organizar y gestionar grandes cantidades de información de forma estructurada y segura. 

Gracias a este sistema, es posible acceder rápidamente a los datos, modificarlos, analizarlos y mantenerlos en orden para dar soporte a aplicaciones, empresas, instituciones y proyectos de cualquier tipo.

## Tipos de bases de datos

Existen diversos tipos de bases de datos como las **relacionales (SQL)**, las **no relacionales (NoSQL)**, las **orientadas a objetos**, las **distribuidas**..., cada uno adecuado para necesidades concretas en el ámbito informático y empresarial.

Nosotros nos vamos a centrar en las primeras (**relacionales o SQL**) debido a que son las más utilizadas debido a  su **fiabilidad y estructura clara**, aunque las **no relacionales o NoSQL** están creciendo en los últimos años debido a la necesidad de acceder a los datos con la máxima celeridad, derivado de la aparición de grandes plataformas como Amazon, Facebook, Instagram...
